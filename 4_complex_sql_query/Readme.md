# SQL-script - complex SQL query combining multiple simulation outputs

This folder contains a SQL-query example that asks the database to return buildings that do not comply to solar access and noise thresholds and whose "class" is "habitation" (i.e., residential buildings) or "healthcare" (e.g., hospitals).

<br>
<br>

The figure below showcases the highlighted buildings failing to comply to the solar access and noise threshold values included in the SQL-query that was executed in the 3DCityDB environment. The results of the SQL-query where exported from 3DCityDB as KML/COLLADA files and imported to Google Earth Pro.
